# Christmas Website 

## Description

The purpose of this website is to provide new or existing people who plan to celebrate Christmas information about what songs to listen to, what foods to make, what decorations to get. 

## Getting Started

### Build Command

* 'npm i'

### Start Command 

* 'yarn start'
